Design Tweaks · Before / After

对照 kimi.com/code 页面风格的小处调整预览。右侧为建议方案，未写入 design-system.html。

01

主 CTA 按钮：蓝 → 近黑

页面里最强的按钮「安装 Kimi Code」是 rgba(0,0,0,.9) 黑底，蓝色只留给链接与高亮。黑色更显克制、高级；蓝色回归「语义强调」角色。

after · 建议
02

展示标题：800 → 600，尺寸略增

大号 + semibold 比 extra-bold 更精致、不「喊」。页面 H1 实测 48px / 600。

03

公告药丸：蓝边蓝字 → 无边框软底

页面同款是 rgba(23,131,255,.1) 淡底、无边框、深色字 + 蓝色链接，圆角 8px，视觉噪音更低。

04

新增：分段控件 Segmented Control

页面用了两次（Terminal/IDE、包月/包年）。浅灰轨道 + 选中态白底微阴影。适合模式 / 视图切换。

现状无此基元（只有 Button 组 / tab）
new · 建议
05

新增：命令 + 复制条

黑按钮 + 浅灰圆角框 + mono 命令 + 复制图标，可复用于安装 / 引导。介于「按钮」与「代码块」之间的轻量形态。

现状有 .p-code / .code，无此内联命令条
new · 建议

06

页面底色：纯白 → 极淡冷灰（可选）

页面实测 #f9fbfc。差别极微妙，仅在想更柔和时考虑；聊天工具纯白也合理，倾向不动。
