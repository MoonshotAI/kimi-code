Add Astron Coding Plan provider with embedded models and settings. Set KIMI_CODE_EXPERIMENTAL_XUNFEI_CODING_PLAN=1, then run /login to select Astron, or configure a provider with type = "astron" in config.toml. The /settings Astron panel controls stream, temperature, max tokens, and web search for Astron models.
