B · Worktrees 面板（dashboard / rich list）

全宽一个页面，每个 worktree 一张卡片，里面展开它的 session、分支 / PR / 脏净 / ahead-behind / 路径 / 操作。侧边栏只做轻量切换。比看板轻、更直接，不一定要拖拽。

C · Chat 标题区的 workspace 可点击

当前 session 所在 worktree 在 chat 标题区其实是一个「闲置」的入口。把它做成可点击的 chip：点击展开一个快捷菜单（查看全部 worktree / 切换 worktree / 新建 worktree / 打开 PR）。让「当前 session 属于哪个 worktree」既是展示，也是入口。
